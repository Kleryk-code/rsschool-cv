_Count the number of divisors of a positive integer n.

Random tests go up to n = 500000._

```
function getDivisorsCnt(n){
     let countDivisor = 0;
     for (let i = n; i > 0; i--) {
       if(n % i === 0) countDivisor ++
     }
     return countDivisor;
}
```
